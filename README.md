# E-Commerce Sales Dashboard — Power BI

## Overview
Interactive dashboard built on e-commerce dataset with 4 pages.

## Pages
- Executive Summary — KPIs, Revenue trends, Map
- Sales Analysis — Category & monthly trends  
- Product Performance — Top 10 products, Scatter chart
- Customer Insights — Membership tier analysis

## Tools Used
- Power BI Desktop
- DAX Measures
- Data Modeling (5 tables)

## Dataset
- 350 Orders, 100 Customers, 52 Products, 8 Categories
